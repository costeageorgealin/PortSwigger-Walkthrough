<h1> File path traversal, simple case </h1> </br>
This lab contains a path traversal vulnerability in the display of product images.
To solve the lab, retrieve the contents of the /etc/passwd file.

Open a random image to access its file path, intercept the traffic, send it to the Repeater, and add ../../../../ to the path.
