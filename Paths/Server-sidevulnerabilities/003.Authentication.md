Lab: Username enumeration via different responses </br>

This lab is vulnerable to username enumeration and password brute-force attacks. It has an account with a predictable username and password, which can be found in the following wordlists: </br>

Candidate usernames </br>
Candidate passwords </br>

To solve the lab, enumerate a valid username, brute-force this user's password, then access their account page. </br>

Proof Of Concept </br>

I initiated the lab by navigating to the 'My Account' section, where I submitted a test login attempt and intercepted the HTTP request via Burp Suite. Using the Intruder tool, I conducted a brute-force attack against the username field using the provided wordlist.  </br>
